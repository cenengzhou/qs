Excel Header must follow the below sequence. (Column A to T)
BQ (Column A to H)
  • BQ Header: Keep all columns blank besides "Description" (Column D).
  • Bill (Column A) : first 2 digits refer to Bill No. while the rest will be the Sub-bill No.
  • Bill, Page, Item should be unique.
  • Description, Quantity, Units, Rate, Value must not be blank for BQ item.
Resources (Column I to T)
  • Resources must refer to a BQ item.
  • Resource Code, Type, Description, Total Qty, Units, Rate, Total must not be blank.
  • Object, Subsidiary code should be provided if they cannot be found in Resource Code Matching Table.
  • Package No (Column T) can be manually input now and must follow the rules below.
    • Package No. should start with '6' for object code '13xxx' (Material Package).
    • Package No. should start with '3' for object code '14xxx' and subsidiary code '3xxxxxxx' (NSC).
    • Package No. should start with '1' or '2' for object code '14xxx' (DSC, NDSC).
