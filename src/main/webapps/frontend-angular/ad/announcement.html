    Form of Subcontract Payment Terms Retention
    Major QS2
    Maximum: 5%, Interim: 10%, MOS: 10%
    (based on Revised Subcontract Amount)
    Minor QS2
    Maximum: 5%, Interim: 10%, MOS: 10%
    (based on Revised Subcontract Amount)
    Consultancy Agreement QS4 No Retention
    Internal Trading QS1 No Retention

System Update (Posted on 20th Jul, 2017)

  • Subcontract Award: Form 4 will not be displayed for Nominated Subcontract (NSC).

  • Main Contract Certificate : Attachment will no longer be able to be deleted when Main Contract Certificate has been posted to Finance or under Finance review.

System Update (Posted on 8th Jun, 2017)

  • Main Contract Certificate: Be able to update Retention Release Schedule on Main Contract Certificate Dashboard.

  • Main Contract Certificate : Be able to remove unused Main Contract Certificate which has not been sent out IPA yet.

System Update (Posted on 4th May, 2017)

  • Main Contract Certificate: A new workflow of Main Contract Certificate has been implemented.

  • Main Contract Certificate Enquiry: has been added under "Enquiry" --> "Job".

System Update (Posted on 13th Apr, 2017)

  • Transit: A new workflow of Transit has been implemented.

System Update (Posted on 16th Feb, 2017)

  • Addendum: Be able to add Subsidiary Code 3999 9999 & 3980 9999 as VO in Repackaging for NSC use

  • User Preference: Be able to customize columns, order and width for the grids

System Update (Posted on 7th Jan, 2017)

  • Subcontract Award: Be able to add subcontractor that doesn't exist in system (not allow to be awarded)

  • Job Cost Enquiry: Be able to search for Posted/Un-Post/All transactions and search by Date Range

  • Calendar for Addendum/Payment/Main Cert: Be able to press on the calendar and jump to the certificate
